# NeuralOS Assets

This directory contains visual assets for documentation.

## Required Assets

| File | Description | Size |
|------|-------------|------|
| `neuraos-logo.svg` | Vector logo | 120x120 |
| `neuraos-banner.png` | Header banner | 800x200 |
| `demo.gif` | Terminal demo | 700x400 |
| `web-panel.png` | Web UI screenshot | 600x400 |
| `footer-wave.svg` | Footer decoration | 1440x100 |

## Creating Assets

### Banner
Use any design tool (Figma, Canva, etc.) with:
- Gradient background (#667eea to #764ba2)
- "NeuralOS" text with glow effect
- Subtle circuit/neural pattern

### Demo GIF
Record terminal session showing:
```bash
# Boot sequence
# Login
# System commands (uname, free, ps)
# Python + NumPy demo
```

Use `asciinema` or `terminalizer` for recording.
